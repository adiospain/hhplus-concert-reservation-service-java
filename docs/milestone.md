    - 플로우 차트
      - 시퀀스 다이어그램을 바탕으로 시작과 종료를 명시하며, 전반적인 프로세스의 흐름을 파악합니다.

      - 필요한 컬럼을 추려내어 테이블을 작성합니다. 
    - API 명세
      - 각 엔드포인트의 기능과 입력 출력 파라미터, 오류 처리를 기술합니다.
